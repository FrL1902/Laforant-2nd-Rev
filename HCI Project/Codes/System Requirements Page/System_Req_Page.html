System Requirements

Minimum Requirements

  • Operating System: Windows 8.1 64 Bit, Windows 8 64 Bit, Windows 7 64 Bit Service Pack 1
  • Processor: Intel Core i5-7400 @ 3GHz (4 cores) / AMD Ryzen 3 2200G (4 cores) @ 3.5GHz
  • Memory: 4GB
  • Video Card: NVIDIA GTX 750 Ti 2GB/ AMD Radeon R7 265
  • Sound Card: DirectX 10 compatible
  • HDD Space: 12GB

Recommended Requirements

  • Operating System: Windows 10
  • Processor: Intel Core™ i5-10400F @ 2.90GHz (6 cores) / AMD Ryzen™ 5 3600 (6 cores) @ 3.6GHz
  • Memory: 8GB
  • Video Card: NVIDIA GTX 1060 / AMD Radeon RX 580
  • Sound Card: DirectX 11 compatible
  • SSD Space: 20GB
